  • Инструменты просмотра


    Статьи доступны в формате PS (PostScript) или PDF.

    • Для чтения статей в формате PS под MS Windows необходимо установить GhostScript и GhostView.
    • Для чтения статей в формате PDF под MS Windwos необходимо установить Evince или Adobe Acrobat Reader.

  • Инструменты редактирования


    Верстка журнала ведется с помощью макропакета LaTeX из дистрибутива MikTeX с использованием собственного стилевого класса. В качестве редактора для осуществления корректуры статей используется TexMaker.

  • Ссылки

    • GhostScript - пакет работы с PostScript.
    • GhostView - просмотрщик PS файлов.
    • Evince - свободный и бесплатный просмотрщик PDF файлов под MS Windwos и Linux.
    • Adobe Acrobat Reader - проприетарный бесплатный просмотрщик PDF под MS Windows.
    • MikTeX - дистрибутив издательской системы TeX, содержит набор утилит для обработки текста.
    • TexMaker - редактор для текста в формате LaTeX. Работает совместно с MikTeX.
