  • Viewers


    Articles are available in PS (PostScript) or PDF format.

    • To read articles in the PS format for MS Windows, you need to install GhostScript and GhostView.
    • To read the articles in PDF format under MS Windwos, you need to install Evince or Adobe Acrobat Reader.

  • Editing Tools

    The layout of the journal is conducted using the LaTeX macro package from the distribution MikTeX using its own style class. As an editor for the implementation of proofreading articles, TexMaker is used.

  • References

    • GhostScript - PostScript work package.
    • GhostView - a PS file viewer.
    • Evince is a free and free PDF file viewer for MS Windwos and Linux.
    • Adobe Acrobat Reader - a proprietary free PDF viewer for MS Windows.
    • MikTeX - distribution kit of the TeX publishing system, contains a set of utilities for processing text.
    • TexMaker is the editor for text in LaTeX format. Works in conjunction with MikTeX.
